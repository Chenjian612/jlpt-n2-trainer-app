❯ Leitner 设计。是啥                                                                                                                                                                       

● Leitner 间隔复习系统
                                                                                                                                                                                           
  是一套用"盒子"模拟遗忘曲线的复习算法，1970年代由德国人 Sebastian Leitner 发明。
                                                                                                                                                                                             ---
  核心思路                                                                                                                                                                                 

  想象有 5 个盒子，编号越大、复习越稀：

  Box 1 → 每天复习
  Box 2 → 每2天
  Box 3 → 每4天
  Box 4 → 每8天
  Box 5 → 毕业（掌握）

  - 答对 → 卡片升一格（Box 1 → Box 2）
  - 答错 → 卡片退回 Box 1，重新开始

  ---
  用在这个 App 里是什么意思

  目前 App 的问题：每次练习题目是随机或按序的，不管你这道题上周答对了 10 次还是刚错过，下次出现概率一样。

  加了 Leitner 之后：
  - 刚错过的词/题 → 明天就会再出现
  - 已经稳定掌握的 → 一周后才会再出现
  - 练习时间花在真正需要的地方

  ---
  对这个 App 的改动范围

  ┌──────────┬──────────────────────────────────────────────────────────┐
  │    层    │                        需要改什么                        │
  ├──────────┼──────────────────────────────────────────────────────────┤
  │ 数据模型 │ ProgressState 新增每道题的"当前 box 编号 + 下次复习日期" │
  ├──────────┼──────────────────────────────────────────────────────────┤
  │ 服务层   │ progressService 答对/错时更新 box                        │
  ├──────────┼──────────────────────────────────────────────────────────┤
  │ UI       │ 首页推荐"今日到期复习"，新增一个复习 mode                │
  └──────────┴──────────────────────────────────────────────────────────┘

  ---
  简单说：让 App 记住你什么时候该复习什么，而不是每次随机出题。Anki 用的就是这个原理的升级版（SM-2算法）。
