## 五、数据架构（JSON 化已完成）

本次 commit d934e8b 完成了 Seed 数据 JSON 化（评审项 C4），各加载器模式如下：

```
src/data/seed/
├── drill_questions.json          ← drillQuestions.ts 读取
├── reading_passages.json         ← readingPassages.ts 读取
├── listening_cases.json          ← listeningCases.ts 读取
├── grammar_study_items.json      ← studyPacks.ts 读取
├── n2_vocab_base.json            ← extendedVocabLibrary.ts 读取
└── official_vocab_decks.json     ← officialVocabDecks.ts 读取
```

优点：内容与代码解耦，无需重新编译即可更新题目。
