# JLPT N2 Trainer App

一个面向 JLPT N2 备考的 Expo / React Native 训练应用。当前版本已经覆盖首页推荐、刷题、记忆包、读解、听力、错题回收和本地进度记录，适合日常碎片化练习与阶段性复盘。

当前这版已经完成一轮主界面产品化升级：首页、刷题、记忆包、读解、听力等主路径在视觉层级、反馈结构和移动端可读性上都做了统一。

## 当前能力

- 首页根据当天完成情况和错题积压，动态推荐下一轮训练。
- 支持 8 个训练模式：
  - 文法闯关
  - 文法记忆包
  - 词汇刷题
  - 词汇记忆包
  - 读解实战
  - 听力要点拆解
  - 文法错题回收
  - 词汇错题回收
- 首页、刷题、记忆包、读解、听力页已完成一轮统一视觉升级，训练主路径的标题、反馈和行动按钮保持同一套语气。
- 刷题结果会自动写入本地进度；答错的题会自动进入回收队列。
- 错题回收支持优先级排序，优先处理高频错误、待首次回收和该复习的题。
- 听力模式已接入官方公开示例音频改编材料，支持播放、重播、倍速和听后复盘。
- 读解、刷题、记忆包、错题回收都提供结果页与下一步建议。

## 目录结构

- `src/app`：应用根节点、Provider、导航。
- `src/features`：首页、模式详情、刷题、记忆包、读解、听力、错题回收等功能页。
- `src/domain`：领域模型与进度/推荐服务。
- `src/data`：seed 数据、题库、音频材料、仓储与存储封装。
- `src/components`：通用 UI 组件。
- `src/theme`：颜色、字体、圆角、阴影等设计 token。

## 本地运行

安装依赖：

```bash
npm install
```

启动开发服务：

```bash
npm run start
```

常用目标：

```bash
npm run web
npm run android
npm run ios
```

如果要用 Expo Go 做远程预览，请使用 Expo 默认的 Metro 端口 `8081` 并配合 `tunnel` 模式。

## 数据与状态

- 训练进度保存在本地 `AsyncStorage`。
- 存储键为 `jlpt-n2-trainer-state-v1`。
- 记录内容包括：
  - 每日训练轮次
  - 每个模式的完成情况
  - 错题列表与最近误选
  - 错题回收状态
  - 周目标与连续天数相关统计

## 已做验证

当前版本已经做过一轮浏览器级手动/自动化检查，确认通过的路径包括：

- 首页推荐与模式进入
- 文法刷题 -> 连续答题 -> 结果页展示 -> 返回首页建议
- 文法刷题 -> 生成错题 -> 错题回收 -> 返回首页
- 文法记忆包进入与翻卡
- 读解进入与提交
- 听力音频加载、播放、提交前解锁逻辑
